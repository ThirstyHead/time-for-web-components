  22. HTML offers a fixed set of elements

    What if you need a new semantic element?

    • CSS id and class allows you to define new semantic styles
    • HTML div and span allows you to group semantically-related content together

  25. #protip

    Anywhere you are using a <div>, <span>, class, or id is a likely candidate for a new Web Component.

  29. Web Components ===
    Encapsulation

  30. Historically, web development has been pathologically global
    (or more charitably, page-centric):

    • One flat level of HTML elements
    • CSS is a global DOM query language
    • All JavaScript is loaded into the global namespace

  31. Modern Web Development is now leaning towards privacy

    • HTML offers Shadow DOM
    • JavaScript let variables are lexically scoped
    • JavaScript modules are private by default (export, import)
